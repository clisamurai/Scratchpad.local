Intro to Scratchpad.local

Hello and welcome to an incredible new notepad/word processor for the productive mind. Scratchpad.local is a browser-based word processor that can be embedded in your bookmarks bar instead of being a standalone app. Scratchpad.local embeds the application with each file you save making sure your recipients receive the file as you intended it and in the correct and current version of Scratchpad.local.

Simply copy the following code into a new tab and bookmark it in any modern browser:

data:text/html, Scratchpad.local

Directions: Press Ctrl/Cmd+S to save | Ctrl/Cmd + O to open | Ctrl/Cmd + P to print
You are viewing, editing or creating a document in Scratchpad.local. There is no autosave functionality. Documents save as HTML files and can be opened in any modern browser.
